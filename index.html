Projeto Castanhas

Exportação para a União Europeia.

O projeto Castanhas foi criado com o intuito de colaborar com a exportação de castanhas produzidas na região norte do Brasil por produtores familiares, de pequeno e grande porte, para a União Europeia, a qual é uma região do mundo que tem grande receptividade pelas castanhas brasileiras.

Projeto Hospitais

Importação de produtos hospitalares alemães.

Este projeto foi criado com o intuito de importar produtos e equipamentos hospitalares alemães de alta qualidade para hospitais do sudeste e sul do Brasil. Os bens importados tiveram boa receptividade entre os médicos e operadores e têm contribuído para o tratamento e diagnóstico de diversas doenças.

Projeto Soja

Exportação de soja para a Ásia.

O continente asiático é um grande consumidor de soja brasileira por conta da quantidade produzida e da qualidade desse produto. Com esse intuito, este projeto foi criado para colaborar com a acessibilidade desse mercado para produtores de Lucas do Rio Verde, no estado do Mato Grosso.
